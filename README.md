# EmoTranslator

A simple Streamlit app to translate between text and emojis.

## Features
- Translate text to emoji and emoji to text
- Clean, minimal interface
- Copy results to clipboard
- Keeps a short translation history

## Skills Demonstrated
- Python programming
- Streamlit web app development
- Basic UI/UX
- Git version control

## Getting Started
1. Install Streamlit:
   ```bash
   pip install streamlit
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Why this project?
To practice building web apps with Streamlit and have fun with emojis.
